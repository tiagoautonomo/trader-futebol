# BetRadar BR ⚽

Aplicativo web (PWA) de **análise de jogos e comparação de odds** das casas de apostas.
Mostra os jogos do dia, compara as odds de várias casas, aponta as apostas de maior
**valor esperado (EV+)**, sugere o valor a apostar pelo **Critério de Kelly** e guarda um
**histórico** com sua taxa de acerto, lucro e ROI.

## Como usar

1. Abra o site.
2. Clique em **⚙️ Dados** e cole sua chave gratuita da [The Odds API](https://the-odds-api.com)
   (500 consultas/mês grátis) para ativar odds reais ao vivo. Sem chave, o app roda em
   **modo demonstração** com dados simulados.
3. Informe sua **banca** e veja as melhores apostas do dia.

A chave fica salva **apenas no seu navegador** (localStorage) — não é enviada a lugar nenhum.

## Instalar como aplicativo

O site é um PWA: no Chrome/Edge, clique em **⬇️ Instalar app** (ou no menu → "Instalar")
para ter ícone na tela inicial e uso offline.
